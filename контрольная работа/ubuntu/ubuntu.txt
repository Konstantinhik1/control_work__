1. Используя команду cat в терминале операционной системы Linux, создать 
два файла Домашние животные (заполнив файл собаками, кошками, 
хомяками) и Вьючные животными заполнив файл Лошадьми, верблюдами и 
ослы), а затем объединить их. Просмотреть содержимое созданного файла. 
Переименовать файл, дав ему новое имя (Друзья человека).

 Создаем файл "Домашние животные" и заполняем его:

cat > Домашние_животные.txt
собаки
кошки
хомяки
^C

 Создаем файл "Вьючные животные" и заполняем его:

cat > Вьючные_животные.txt

ослы
^C

 Объединяем два файла в один:

cat Домашние_животные.txt Вьючные_животные.txt > Друзья_человека.txt

cat Друзья_человека.txt

 Переименовываем файл, давая ему новое имя:

mv Друзья_человека.txt Друзья_человека.txt

3.Подключить дополнительный репозиторий MySQL. Установить любой пакет
из этого репозитория.

 Обновляю индексы пакетов:

sudo apt update

 Устанавливаю пакет, необходимый для добавления репозитория MySQL:

sudo apt install software-properties-common

 Добавляю репозиторий MySQL:

sudo apt-add-repository 'deb http://repo.mysql.com/apt/ubuntu/ bionic mysql-5.7'

Заменяю bionic на название моего дистрибутива Ubuntu 

 Обновляю индексы пакетов, чтобы система знала о новом репозитории:

sudo apt update

Установить и удалить deb-пакет с помощью dpkg

 Устанавливаю пакет mysql-client:

sudo apt install mysql-client


4.Установить и удалить deb-пакет с помощью dpkg

 Устанавил пакет с помощью dpkg:

sudo dpkg -i mysql-server_8.0.30-1ubuntu20.04_amd64.deb

 Проверяю, что пакет установлен:

dpkg -l | grep mysql-server

Вы должны увидеть информацию об установленном пакете.

 Теперь давайте удалим этот пакет:

sudo dpkg -r mysql-server

Эта команда удалила пакет mysql-server с моей системы.

 Проверяем, что пакет удален:


dpkg -l | grep mysql-server
